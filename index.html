Политика обработки персональных данных Дата последнего обновления: 18.11.2024 1. Общие положения Настоящая Политика обработки персональных данных (далее — «Политика») определяет порядок обработки персональных данных пользователей мобильного приложения "МедЭциклопедия" (далее — «Приложение»). 2. Сбор и обработка персональных данных Приложение "МедЭциклопедия" не осуществляет сбор, хранение, обработку или передачу каких-либо персональных данных пользователей. 3. Функционирование приложения Приложение представляет собой автономную медицинскую энциклопедию, которая: Работает полностью в автономном режиме Не требует регистрации пользователей Не собирает информацию о пользователях Не отправляет никакие данные на внешние серверы Не использует файлы cookies Не отслеживает местоположение пользователей 4. Разрешения приложения Приложение запрашивает только базовые разрешения, необходимые для его функционирования: Доступ к интернету (только для перехода по внешней ссылке на Telegram-канал) 5. Безопасность Поскольку приложение не собирает персональные данные, риски, связанные с их обработкой, отсутствуют. 6. Изменения в политике конфиденциальности Мы оставляем за собой право вносить изменения в данную Политику. При внесении изменений в актуальной редакции указывается дата последнего обновления. 7. Контактная информация По всем вопросам, связанным с данной Политикой, вы можете связаться с нами через Telegram-канал: https://t.me/doc_talks 8. Заключительные положения Используя Приложение, вы подтверждаете, что ознакомились с данной Политикой. Настоящая Политика действует бессрочно с момента ее опубликования.
